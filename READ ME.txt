Go into assets folder and drag editor.swf into flashplayer_32_sa.exe.

If you're using Mac, go into assets folder and drag editor.swf into flashplayer_32_sa.dmg.

If you're using Linux, go into assets folder, extract flash_player_sa_linux.x86_64.tar.gz to same folder, and drag editor.swf into flashplayer.
